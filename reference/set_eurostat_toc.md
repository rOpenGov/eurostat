# Set Eurostat TOC

Internal function.

## Usage

``` r
set_eurostat_toc(lang = "en")
```

## Arguments

- lang:

  2-letter language code, default is "`en`" (English), other options are
  "`fr`" (French) and "`de`" (German). Used for labeling datasets.

## Value

Empty element
